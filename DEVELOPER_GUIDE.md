# Developer Guide

## Initial Setup

Install dependencies

```bash
make setup
```

## Building

To build locally

```bash
make
```

To see the help targets

```bash
make help
```

I use some workflows in github actions (./github/workflows) to compile and test.  

## PRs

Use github PRs and vefify using "Compile and Test" actions that the PR is good.
